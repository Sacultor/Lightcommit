# Web3.Storage API密钥获取指南

## 步骤1：注册账户

1. 访问 [Web3.Storage](https://web3.storage/)
2. 点击右上角的"Sign up"或"Sign in"按钮
3. 使用你的邮箱注册账户或登录
4. 验证你的邮箱地址（如果需要）

## 步骤2：创建API令牌

1. 登录后，点击右上角的用户头像或菜单
2. 选择"API tokens"或"API密钥"选项
3. 点击"Create new token"或"新建令牌"按钮
4. 输入令牌名称（例如："Lightcommit项目"）
5. 选择令牌权限（通常选择默认权限即可）
6. 点击"Create token"或"创建令牌"按钮

## 步骤3：复制令牌

1. 创建成功后，系统会显示你的API令牌
2. 点击复制按钮复制令牌（通常以"ey"开头的长字符串）
3. 立即将令牌保存到安全的地方，因为出于安全原因，令牌只显示一次

## 步骤4：配置到项目中

1. 打开项目根目录下的`.env`文件
2. 找到以下行：
   ```
   WEB3_STORAGE_TOKEN=
   ```
3. 在等号后面粘贴你复制的令牌：
   ```
   WEB3_STORAGE_TOKEN=ey...（你的完整令牌）
   ```
4. 保存文件

## 注意事项

1. **安全第一**：API令牌是敏感信息，不要分享给他人
2. **不要提交到版本控制**：确保`.env`文件在`.gitignore`中，不会被提交到代码仓库
3. **定期更换**：为了安全，建议定期更换API令牌
4. **权限最小化**：只授予令牌必要的权限

## 验证配置

运行以下命令验证配置是否正确：
```bash
npm run test:ipfs
```

如果看到"Web3.Storage令牌格式正确"的绿色提示，说明配置成功！

## 常见问题

**Q: 我忘记了令牌，怎么办？**
A: 你需要回到Web3.Storage网站，删除旧令牌并创建一个新令牌。

**Q: 令牌有使用限制吗？**
A: 是的，免费账户有一定的存储和带宽限制，详情请查看Web3.Storage的定价页面。

**Q: 我可以在多个项目中使用同一个令牌吗？**
A: 可以，但为了更好的管理和安全，建议为每个项目创建单独的令牌。
